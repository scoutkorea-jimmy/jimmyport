        Deploy · GitHub 바로 커밋

        이 페이지에서 GitHub Personal Access Token(PAT)을 입력하면 data/site-config.json 과 data/portfolio.json 을 저장소에 직접 커밋합니다. PAT은 이 브라우저의 sessionStorage에만 머무릅니다.

        PAT 만드는 방법
        1. github.com → Settings → Developer settings → Personal access tokens → Fine-grained tokens → Generate new token
        2. Repository access: jimmyport 단일 저장소만
        3. Repository permissions → Contents: Read and write
        4. 만들어진 github_pat_… 토큰을 위 입력란에 붙여넣으세요.
        5. 토큰은 이 브라우저의 sessionStorage에만 남고, 탭을 닫으면 사라집니다.
