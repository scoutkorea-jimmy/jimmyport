# Jimmy Park Homepage

Static homepage for [jimmypark.net](https://jimmypark.net/).

## Files

- `index.html`: homepage markup and metadata
- `styles.css`: responsive visual design
- `script.js`: small runtime enhancements
- `CNAME`: custom domain for GitHub Pages

## Portfolio

Use `portfolio-admin.html` to draft YouTube-based portfolio items. Download the
generated `portfolio.json`, then replace `data/portfolio.json` and deploy.
